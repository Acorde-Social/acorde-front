# Acorde Frontend

Frontend da plataforma Acorde para colaboração musical entre compositores e músicos.

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- Socket.io (para notificações em tempo real)
- Jest (para testes)

## Instalação

```bash
# Instalar dependências
yarn install

# Iniciar servidor de desenvolvimento
yarn dev
